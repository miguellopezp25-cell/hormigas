# BlueAnts 🐜

Tienda especializada en colonias de hormigas vivas y hormigueros profesionales.

## Stack

- **Next.js 16** — App Router, SSG/SSR
- **TypeScript**
- **Tailwind CSS 4**
- **shadcn/ui** — Radix UI components
- **React 19**

## Rutas

| Ruta | Descripción |
|---|---|
| `/` | Landing page |
| `/hormigas` | Catálogo de especies |
| `/hormigas/[id]` | Detalle de especie |
| `/hormigueros` | Catálogo de hormigueros |
| `/hormigueros/[id]` | Detalle de hormiguero |
| `/blog/[slug]` | Artículos y guías |

## Desarrollo

```bash
pnpm dev
pnpm build
pnpm lint
```
